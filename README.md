## Sparkchain Ai Bot 脚本怎么使用?
1、config目录里面包含``accounts.json`` 和 ``proxy.txt``

``accounts.json`` 格式
```json
[
    {
        "Email": "email 1",
        "Password": "password 1"
    },
    {
        "Email": "email 2",
        "Password": "password 2"
    }
]
```

``proxy.txt`` 格式一行一个或者由脚本在代理池抽取
